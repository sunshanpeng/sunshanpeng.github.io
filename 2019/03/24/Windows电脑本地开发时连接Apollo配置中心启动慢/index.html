Windows电脑本地开发时连接Apollo配置中心启动慢

三月 24, 2019 java 本文总阅读量次

表现：

启动时中间停顿了10秒左右

原因：

电脑上网卡太多

解决方法：

  1. 禁用多余网卡

  2. 在启动参数指定本机IP：-Dhost.ip=10.208.202.251

    10.208.202.251改成自己的IP地址

原理：

指定本机IP后就会跳过查找有效IP这一步
